This experiment is used to measure the performance of IPOS versions.

The measurement metric:
	time-to-compelation. 

The measurement tool: 
	Logic 16 (a logic analyzer from www.saleae.com)

The hardware to be used:
	TI launchpad MSP430FR5969 (mcu: MSP430FR5969)

The applications to be tested are: 
	1- Data_decompression 
		The Data_decompression application will attempt to decompres 
		100 bytes 

	2- sortAlgo (selecton sort algorithm)
		The sortAlgo applicaiton will attempt to sort 20 numbers
	3- largeDataCommit: 
		Manipulate and commit 50 variables
